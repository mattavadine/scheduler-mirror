2.3.3 Manage Classrooms

The Manage Classrooms dialog lets the Administrative Scheduler add, remove, and update a classroom to the Room Database. The scenarios below describes the functions of adding, removing, and updating (editing) a room's information in the database. Below is a picture of the intial Manage Classrooms dialog.

Figure: Intial Manage Classrooms Dialog

In the upper left section of the dialog window there is a 3 column table that contains a list of building numbers, corresponding building names, and room numbers. The box above the name column is a search bar that allows the user to search for a specific building number, room number, or name. If the user selects a specific room from the table the Room Availability calender will be populated from the database. The Room Availability calender shows which days the room is occupied by highlighting the cell red. On the bottom of the building list, the Manage Classrooms dialog provides two action button. The Add button opens up a dialog which the user can add another room to a specific building. The Remove button lets the user remove the selected room from the corresponding building.

The Manage Classrooms dialog window also provides detailed information of a selected room under the Advanced View tab in the upper left corner. Selecting this tab displays information about all aspects of the selected room. The Building and Room labels contain the building number and the room number of that particular room. The Max Capacity label contains the maximum number of students and faculty allowed to occupy the room at any given time. The Disabilities checklist shows the various disability accomadations that the room contains. The Equipment List contains the equipment available to that room. The Room Equipment list contains the equpipment that is physically present in the room. The Room Time Availability spread sheet describes when the room is available and occupied.

2.3.3.1. Add Room

To add a room, the user opens up the Manage Classroom dialog. Once the Manage Classrooms dialog is opened, the user clicks on the Add button. The system then displays the a blank template of the Advanced View dialog to fill out. See Figure below.

figure 38: Add Room.

To add a room, the user fills out the required fields for the Room Record. The user selects which building number the room is in. Then the user inputs the room number of the room and the max capacity for that particular room. The user also selects the type for the room, i.e. (Lab and/or Lecture). The user can then check any of the listed disabilities to include additional equipment to accomodate. Next, the user adds an equipment from the Equipment List to the Room Equipment list by selecting the equipment item (on the left) and clicking on the Add button. The Add button adds the equipment item to the Room Equipment list. To remove equipment from the room, the user selects the desired equipment item from the Room Equipment list (on the right) and clicks on the Remove button. If the user wants to add all the equipment from the Equipment List to the Room Equipment list, he can click on the Add All button. Likewise, if the user wants to remove all the equipment item from the Room Equipment list, he can click on the <<Remove All button. Once the user finishes adding equipment to the room, he can leave any comments or notes, in the Addtional Notes dialog, about the room that need to be addressed. Lastly he can fill out the Room Availability for the particular room. By clicking on an individual cell or dragging through multiple cells, the user can choose when the room is available.

Once the user finishes filling out all the necessary fields, he clicks on the Save button to add the room to the classroom database. If the user wants to cancel his room addition, he can click on the Room Search tab in the upper left corner which clears all fields in the Advanced View dialog window.

2.3.3.2 Edit Classroom

To change or update a room's characteristics, the Administrator selects a particular room from the list of displayed rooms. The user then clicks the Advanced View tab in the upper left corner. The system then displays the information that is related to the selected room. The user can change any of the fields in this view and when finished clicks the Save button to update the room information in the database.

2.4.3.3. Remove Room

To remove an room, the user navigates the the Manage Classroom screen. The user then selects the specific room he wishes to remove from the databse. The user clicks on the Remove button to remove the room from the database. The will be prompted with a confirmation dialog and the user then clicks on the OK button to permanently remove the room. The user clicks on the Cancel button to stop the process of deleting the room from the database.
