Schedule Generation

<-- Prev Top Next -->
The Schedule Setup screen allows the user to specify which courses are included in the generated schedule. If a previous schedule exists, the offered courses list will be populated with the previous schedule's list of offered courses. If no previous schedule exists, all the courses will appear on the master course list. The user then drags the desired courses to the offered courses list or to specific times on the calendar to enforce time constraints. Once a course is moved to the offered course list or the calendar it is removed from the master course list.
