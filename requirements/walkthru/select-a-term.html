After the user inputs a valid username and password on the Login page, the Scheduler will present the user with the Home page as shown above. The page contains a list of previously created schedules. The user can select one and click the 'Open' button to use the selected schedule as a starting template. There is also a 'New Schedule' button that will allow the user to start building a new schedule from scratch.
