Schedule View

This is the default window to view the schedule of a particular selection. The Search Box is where classes, teachers, rooms, or departments are entered in order to be displayed in the table. The filtering button toggles on and off various schedule viewing options. When the table is filled, hovering the mouse over any of the scheduled items will display information about that entery.

When a teacher is searched for in the search box, the table is filled with the classes that teacher is scheduled to teach for the quarter.

When a class room is searched for in the search box, the table is filled with all classes that partake in that particular room.

When a department is searched for in the search box, the table is filled with all the classes that the department is scheduled to teach that quarter. Overlapping classes are split in the boxes and assiagned colors to distinguish between them.
