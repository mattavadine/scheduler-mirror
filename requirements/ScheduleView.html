Schedule View

This is the default window to view the schedule of a particular selection. The Search Box is where courses, instructors, rooms, or departments are entered in order to be displayed in the table. The filtering button toggles on and off various schedule viewing options. When the table is filled, hovering the mouse over any of the scheduled items will display more information about that entery. Selections are colored to help distinguish them from empty timeslots.

When an instructor is searched for in the search box, the table is filled with the classes that instructor is scheduled to teach for the quarter. Each class is represented in a box in the table. The size of the box is relevent to the times listed on the left side of the table. If the box is in the timeslot 8:00-9:00, then the class takes place from the hours of 8:00 to 9:00. In the box, information for the classes that instructor teaches is listed. The ordering for the class is DEPT CLASSNUMBER - LEC/LAB Example: CHEM 124 - Lec

When a specific room is searched for in the search box, the table is filled with all courses that partake in that particular room. Each class is once again represented in a colored box in the table. The size of the box corresponds the times listed on the left size of the table. If the box lies in the selection marked 8:00-9:00, then the course partakes between those hours. The content of the box shows the course offered, the type of class, and the instructor teaching. The format is DEPT CLASSNUMBER - LEC/LAB INSTRUCTORNAME Example: CHEM 124 - Lab Kingsbury

When a department is searched for in the search box, the table is filled with all the courses that the department is scheduled to teach that quarter. Overlapping classes are split in the boxes and assiagned different colors to distinguish between them. As much information as possible is diplayed within the boxes. The information displayed is the course, the type of course, the room, and the instructor teaching. The format goes as follows DEPT CLASSNUMBER - LEC/LAB Rm: ROOM INSTRUCTOR NAME Example: CHEM 124 - Lec Rm: 52-107 Kingsbury

When hovering over a selection that isn't displaying all the information noted above, an information box will arrive and diplay all the information, in the same format as above, for the entry.
