2.3: Database Management

When the administrator clicks the Schedule drop down, the class, instructor, and room fields appear. Each link leads to a database window where information can be added, edited, or removed for each database. Section 2.1.1 Figure 2 illustrates this process. The database has the permanent data for instructors, rooms, and courses.

Once the user clicks any of these buttons, the Scheduler displays the database window. Data modified in this section makes a permanent change to the database unless otherwise specified.

The scenarios in this section are broken up into three sections.

      -Instructor Database
      -Course Database
      -Location Database
