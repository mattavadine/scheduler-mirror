2.3.3 Manage Classrooms

The Manage Classrooms dialog lets the Administrative Scheduler add, remove, and update a classroom to the Room Database. The scenarios below describes the functions of adding, removing, and updating (editing) a room's information in the database. Below is a picture of the intial Manage Classrooms dialog.

Figure 1: Intial Manage Classrooms Dialog (Unpopulated)

In the upper left section of the dialog window there is a 3 column table that contains a list of building numbers, room numbers, and corresponding building names. The box above the Name column is a search bar that allows the user to search for a specific building number, room number, or building name. If the user selects a specific room from the table the Room Availability calender will be populated from the database. The Room Availability calender shows which days the room is occupied by highlighting the cell red. On the bottom of the building list, the Manage Classrooms dialog provides two action button. The Add button opens up a dialog which allows the user can add another room to a specific building. The Remove button allows the user remove the selected room from the corresponding building.

Figure 2: Filled Manage Classrooms.

The Manage Classrooms dialog window also provides detailed information of a selected room under the Advanced View tab in the upper left corner. Selecting this tab displays information about all aspects of the selected room. The Building and Room labels contain the building number and the room number of that particular room. The Max Capacity label contains the maximum number of students and faculty allowed to occupy the room at any given time. The Disabilities checklist shows the various disability accomadations that the room contains. The Equipment List contains the equipment available to that room. The Room Equipment list contains the equpipment that is physically present in the room. The Room Time Availability spread sheet describes when the room is available and occupied.

The figure below shows the Advanced View, which displays information about the selected classroom.

Figure 3: Advanced View

2.3.1.1 Add A Classroom

To add a room, the user opens up the Manage Classrooms dialog. Once the Manage Classrooms dialog is opened, the user clicks on the Add button. The system then displays a blank dialog of classroom preferences for the user to fill in.

Figure 4: Add Room Dialog.

To successfully add a room, the user fills out the required fields for the Room Record. The user selects the Building number with the drop down list. Then the user inputs the Room number of the room and the Max Capacity for that particular room. The user also selects the Type for the room, i.e. (Lab and/or Lecture). The user can then check any of the listed Disabilities to include additional equipment to accomodate. Next, the user adds equipment from the Equipment List to the Room Equipment list by selecting the equipment item (a cell on the left) and clicking on the Add button. The Add button adds the equipment item to the Room Equipment list. To remove equipment from the room, the user selects the desired equipment item from the Room Equipment list (on the right) and clicks on the Remove button. If the user wants to add all the equipment from the Equipment List to the Room Equipment list, he can click on the Add All button. Likewise, if the user wants to remove all the equipment from the Room Equipment list, he can click on the Remove All button. Once the user finishes adding equipment to the room, he can leave any comments or notes, in the Addtional Notes dialog, about the room that need to be addressed. Lastly he can fill out the Room Availability for the particular room by first clicking on the Edit. Then by clicking on an individual cell or dragging through multiple cells, the user can choose when the room is occupied.

Once the user finishes filling out all the necessary fields, he clicks on the Save button to add the room to the classroom database. If the user wants to cancel his room addition, he can click on the Room Search tab in the upper left corner which clears all fields in the Advanced View dialog window and switches to the Room Search view (Figure 2).

2.3.3.2 Edit Classroom

To change or update a room's characteristics, the Administrator selects a particular room from the list of displayed rooms (Figure 2). The user then clicks the Advanced View tab in the upper left corner, while the room's cells is highlighted. The system then displays the information that is related to the selected room. The user can change any of the fields in this view and when finished clicks the Save button to update the room information in the database.

Below is a GUI for changing room.

Figure 5: Edit Room

2.3.3.3 Remove Room

To remove a room, the user navigates the the Manage Classroom screen. The user then selects the specific room he wishes to remove from the databse. The user clicks on the Remove button to remove the room from the database. The user will be prompted with a confirmation dialog and the user then clicks on the OK button to permanently remove the room. The user clicks on the Cancel button to stop the process of deleting the room from the database.
