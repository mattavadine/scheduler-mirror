Schedule Generation Drag Accepted

<<  Prev - Top - Next  >>

When a user drags a course onto the calendar, the program verifies that there aren't any conflicts with the proposed schedule. If the schedule does not contain any conflicts, the course is added to the calendar. In this instance, CHEM 252 has been added to the schedule. The course is colored light green to represent that the scheduled time is preferred based on the preferences of an instructor.
