When the user launches the scheduling program, the program displays its start-up screen. The commands along the left of the screen are the main functions of the program. The Home screen is where the scheduling process begins. The Schedule commands allow the user to to Build, View, and Edit a schedule. The Manage commands allow the user to create and edit information for Instructors, Courses, and course Locations. To get to the Home screen, the user logs in.
