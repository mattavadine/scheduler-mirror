Instructor Course Preferences

<<  Prev - Top - Next  >>

When the user presses the 'Courses' button on the preceding screen, the program displays this dialog. For a user acting as department scheduler, the edits can be performed for any instructor. When individual instructors run the scheduling program, they set their own preferences. The instructor-set preferences are the default values used for scheduling. Note that a department scheduler can only edit preferences for a particular schedule, not the instructor's permanent stored preferences.
