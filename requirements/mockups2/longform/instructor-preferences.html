Instructor Preferences

Once an administrator clicks on the any of the instructor preference buttons from an edit instructor view, they are greeted with the respective preference view.



Course Preferences

This view as shown in Figure 1 is reached by clicking on an instructor's Courses button.


Figure 1: Instructor course preferences


This screen will list all courses for the department. Beside each course is a drop down with preference options for that specific course. The options are:

  1. Preferred - I am highly qualified and would really like to teach.
  2. Acceptable - This is not a priority course, but I am qualified and would not mind teaching.
  3. Not Preferred - It would put strain on me, but I could teach if necessary.
  4. Not Possible - There is absolutely no way I can teach this course.

Acceptable is the default choice for all courses.

When the preferences have been set, the user must select the Save button to save the changes. Otherwise, select the Cancel button.



Time Preferences

This view as shown in Figure 2 is reached by clicking on an instructor's Times button.


Figure 2: Instructor time preferences


This screen will show a calendar week view. It allows the user to set an instructor's time preferences. Times are added using the settings at the top.

At the beginning, the week view will be blank indicating a neutral preference for all possible instruction hours. The two text entry boxes allow the user to enter the beginning and ending time of the block they wish to add. The Mon, Tue, Wed, Thu, and Fri checkboxes specify which days of the week for which the block will be added. To the right, above the Add and Delete buttons is a drop down that allows you to choose a preference level from the choices listed below:

  1. Preferred - If there are no other constraints, these are the hours I would like to teach. Will appear as green.
  2. Acceptable - Courses scheduled any time in this range are acceptable. Will appear as yellow.
  3. Not Preferred - I would require special arrangements due to previous commitments. I can do this only if absolutely necessary. Will appear as orange.
  4. Not Possible - There is absolutely no way I can teach during these periods. Will appear as red.

Once the user is done using the available adjustments to add a chunk of time, they can then click on the Add button. Chunks of time added will appear on the week view. For deletion of chunks of time, the user may click on the chunk and then click on the Delete button.

When the preferences have been set, the user must select the Save button to save the changes. Otherwise, select the Cancel button.



Distance Preference

This view as shown in Figure 3 is reached by clicking on an instructor's Distance button.


Figure 3: Instructor distance preference


There are three user input fields on this view. The first input is specifying the preferred center point on the map of the campus for which to apply the distance value given previously. This is done by specifying the building number that the instructor wishes for their courses to be nearest to.

The second input is the numerical value for maximum preferred distance.

The third is a drop down for specifying the distance unit for the value given in the first input. Drop down choices are feet, yards, meters, miles, and kilometers.

However, if the first input is left blank, then the distance value is used to specify how far apart back-to-back classes can be. For example, say the instructor is willing to walk up to 500 feet. If the instructor has three classes (call them A, B, and C, in that order) to teach in three different classrooms, with no break in between, it is possible that the distance between A-B and B-C is 500 feet each and A-C is 1000 feet.

When the preferences have been set, the user must select the Save button to save the changes. Otherwise, select the Cancel button.
