2.7 Installation

The Scheduling Project is a web application. In order to run the application the user simply points their browser to the URL of the appropriate server.

2.7.1 Server Installation Configuration

To setup the Scheduling Project server the user must first install Apache Tomcat. For the next step in the process the user must install and configure MySQL. The user must then run the provided database setup scripts in order to intialize the database. For the last step the user must deploy the WAR.
