2.2 Basic Schedule Creation

 When the user wants to build a brand new schedule, use an existing schedule as a template, or modify a schedule the user must navigate to the Schedule Creation view. In order to navigate to the Schedule Creation user interface, the user must click on the Schedule tab located in the Scheduling tool's header at the top of the application. After clicking this tab the user is presented with the option of selecting a new schedule, using an existing schedule as a starting template, or modify an existing schedule.

2.2.1 Automatic Scheduling

After the user selects the New Schedule option from the Home Screen, the user is presented with the screen show in Figure 2.2.1.1.

Figure 2.2.1.1 Basic Schedule Creation New Schedule

The left-most table consists of a list of available Department Courses. The user can drag courses from the Department Courses table directly onto their preferred time slot on the calendar. If the user prefers that the Scheduling program allocates the course days and times automatically, they can either double-click or drag the course onto the Auto-Generate Courses table on the right. Figure 2.2.1.2 shows the screen after a user has added courses to the Auto-Generate Courses list.

Figure 2.2.1.2 Basic Schedule Creation Automatic SCheduling

Once the user has populated the Auto-Generate Courses table, the user presses the Auto Fill button at the bottom of the Auto-Generate Courses table, to automatically assign all courses found in the Auto-Generate Courses to specific time slots on the calendar.

After the courses in the Auto-Generate Courses table are scheduled into the calendar, the Auto-Generate Courses table is cleared. The courses are displayed in their appropriate time slots and are colored light green, which indicates that the scheduled time is preferred based on the preferences of an instructor. If the user wishes to get more information about a scheduled course, the user hovers the mouse on a specific course and additional information (course title, course number/section, course location, course instructor, schedules course time) will appear in a small text box as shown in Figure 2.2.1.3 for CHEM 461.

2.2.2 Manual Scheduling

Manual Scheduling Valid Drag and Drop

If the user wishes to manually edit the schedule, the user can drag a course from the Department Courses table directly onto a time slot on the calendar. The user clicks and holds down on the specific course and drags the selected course onto the calendar. If the course meets the university requirements and is preferred based on the preferences of the instructor, the course is colored bright green. In Figure 2.2.2.1 the user is attempting to drag CHEM 252 to the 7AM - 9AM Tuesday/Thursday time slot. Once the user unclicks, the course turns light green and is scheduled in the specified time slot and after clicking the Save Schedule button the schedule is saved to the database.

Figure 2.2.2.1 Manual Scheduling Valid Drag and Drop

If the user drags a course onto the schedule that can be offered over different sets of days during the same time slot, the user is presented with a pop-up box of all available offerings. In Figure 2.2.2.2 the user attempts to schedule CHEM 474 from 7AM - 9AM but the course can be offered on both Monday, Wednesday, Friday or Monday, Wednesday, Thursday from 7AM - 9AM. In the example the user selects MWR 7AM - 9AM and presses the Save Schedule button to save it to the database.

Figure 2.2.2.2 Manual Scheduling Day/Time Selection

Manual Scheduling Not Preferred Drag and Drop

If the user wishes to manually edit the schedule, the user can drag a course from the Department Courses table directly onto a time slot on the calendar. The user clicks and holds down on the specific course and drags the selected course onto the calendar. If the course is colored bright orange while attempting to place the course into a specific time slot it means the course is scheduled in a time that is Not Preferred due to the preferences of an instructor. In Figure 2.2.2.3 the user attempts to schedule CHEM 590 from 9AM - 12PM, but the course is not prefered due to the preferences of the instructors in the database. The course is colored bright orange until the user unclicks and places the course in the time slot, then the course is colored light orange.

Figure 2.2.2.3 Manual Scheduling Not Preferred Drag and Drop

Manual Scheduling Invalid Drag and Drop

If the user wishes to manually edit the schedule, the user can drag a course from the Department Courses table directly onto a time slot on the calendar. The user clicks and holds down on the specific course and drags the selected course onto the calendar. If the course is in direct conflict with the scheduling algorithm, university scheduling constraints, or instructor preferences the course is colored red. Once the user attempts to place the course into the time slot the course automatically jumps back to the Department Courses table because it is unable to be scheduled.
