2.6 Contents of 'File' Menu

This section details the additional functionalities available to the administrator from the file menu. A user may create a new schedule, save or load a schedule, import an existing schedule, export a schedule to a file, merge two or more existing schedules, or print a schedule. Figure 2.6.0.1 shows the options located under the file menu.

Figure 2.6.0.1 - File Menu

2.6.1 Creating a New Schedule

A user may create a new schedule by selecting the New option under the file menu. Once the user selects the New option, a blank schedule view will appear with "Untitled" listed under the schedule name.

2.6.2 Opening an Existing Schedule

A user may open an existing schedule by selecting the Open... option under the file menu. Once a user selects the Open... option, an Open Schedule screen will load. The Open Schedule screen will contain a list of all existing schedules in the system. Once the user selects the schedule they want to open from the list, the Open button can be pressed to load the specified schedule. Additionally, the Open Schedule screen will also contain New Schedule and Import buttons in the upper right portion of the screen if the user decides to create a new schedule or import an existing schedule instead of opening an existing schedule in the system.

2.6.3 Saving a Schedule

Both the Save and Save As... options under the file menu provide saving functionality for schedules. In order to use these options the user needs to have a schedule opened. If the open shedule has never been saved before, the Save As... dialog menu will appear if the user selects the Save or Save As... option (See Figure 2.6.3.1). This dialog prompts the user to specify a name to save the schedule as. If the open schedule has been previously saved, the Save option will perform a quick save on the schedule without prompting the user for any information, however, the Save As... option will still prompt the user to enter a name to save the schedule as and will make a new save instance of the schedule under this provided name.

Figure 2.6.3.1 - Save Dialog

2.6.4 Importing a Schedule from File

A user may import a schedule that has been previously exported by selecting the Import... option under the file menu. Once the user selects the Import option, a dialog box appears prompting the user to specify a file to import (See Figure 2.6.4.1). This file will be a previously exported CSV (Comma Separated Value) file. The user selects a file to import, presses the Submit button, and the selected schedule file is imported and displayed.

Figure 2.6.4.1 - Import Dialog

2.6.6 Merging Schedules Together

A user may merge two or more schedules together by selecting the Merge... option under the file menu. Merging schedules allows a user to combine two or more schedules into a single schedule with all the components of the schedules used in the merge. Once the Merge... option is selected, a dialog box will appear prompting the user to select two or more schedules from the list of schedules currently in the system.
