1.5.2. CyberMatrix Class Scheduler

http://www.cybermatrix.com/class_scheduler.html

The CyberMatrix software is built to create schedules easily and effectively for schools and educational institutions. CyberMatrix also produces many other meeting manager, employee scheduling and timesheet applications. This program carrys much of the functionally that would be required for the scheduling software the group is going to create so it is a good application to compare with.

Good Features:
  • Extensive user manual and commonly asked questions database
  • Easy version checker
  • Simple to add new entries
  • Easy to create individualized schedule pdf's for instructors or students
  • Printing options
  • Allows for automatic and manual adjustments to schedules
  • Supports multiple users
  • Supports other languages
  • Data import and export options
  • Security features
  • Incorporated emailing options

Bad Features:
  • Ambiguous data entry boxes
  • Does not support individual class preferences for professors
  • GUI is not as useful as it could be

Missing Features:
  • Linux/Mac support
  • More interactive GUI
Notes:
  • Uses a SQL database
