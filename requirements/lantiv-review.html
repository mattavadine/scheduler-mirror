1.5.5. Lantiv Timetabler

Lantiv Timetabler is a software used for scheduling classes for schools’ administrator. This program has two different mode; interactive and automatic. Under interactive mode, users are able to manually customize the schedule.  When the automatic mode is selected, the program will make a schedule based on the constraints, which are set by users. Lantiv Timetabler is a software that does not have a user’s guide. However, it contains many video tutorials and forums. Overall, Lantiv Timetabler provides an acceptable level of scheduler functionality for elementary schools or middle schools’ class structure.


Good Features:

  • Different view modes; Week, Day, and Personal

  • Schedule can be sorted by teachers, groups, class rooms, students, or equipments

  • Drag/drop customized features

  • Constraint free time, number of classes per day, and start/end time of the day

  • Editable displayed information on each cell

  • Fixated cell feature

  • Forum

  • Video tutorial

  • Blocked option for teacher’s time preferences

  • View report in detailed or statistic mode

  • Group feature that avoids conflicts between classes in the same group

  • View available location or instructor during selected time period


Bad Features:

  • Manually type to insert data such as classes, teachers, and students, into database system

  • Only designed for Windows

  • Set constraint by manually editing each data

  • All classes must be in the same length

  • Difficult to select/insert time

  • Not user-friendly

  • No user manual


Missing Features:

  • Importing database or file

  • Location Proximity

  • Time preferences for instructors

  • Instructor course preferences

  • Security

  • Administration
