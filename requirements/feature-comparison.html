Features:
  1. Ability to control settings through the GUI
  2. Ability to save a report in different file format
  3. Ability to view specific location avalibility throughout the day
  4. Ability to view schedule weekly
WhenToWork:
  1. UI is simple and intuitive. Not incredibly flashy/shiny.
  2. Available only when viewing the schedule "By Employee"
  3. Depends on which of multiple file formats you choose to save as
  4. "Location" --> Job position
  5. This feature could not be found in the available, online demo. However, the program's specifications cite this as a feature
  6. Will highlight time conflicts, but cannot be configured to create schedules to avoid them.
  7. There seem to be only two "groups": Managers and Employees, with separate permissions delegated to each.
  8. While unavailable in such a form, online material seemed sufficient. Furthermore, it is altogether likely that a software purchase would bundle one in.

Cyber Matrix:
    1.  Only allows for the deletion of a class through the graphical schedule calendar.
   2.  Uses a SQL database.
   3.  Includes ability to back up database via email.
Latniv Timetabler:
  1. Users are able to view reports in different view mode
  2. Only .xml file type
  3. Users are allowed to edit the information displays on each cell
  4. Lantiv Timetabler allows users to limit numbers of day per week, the length of each day, and the start time of each day
  5. To avoid overlapping classes, users need to group those classes in one group
  6. Lantiv Timetabler does not provide a user manual. However, there are many video tutorial available online.

Asc Timetables:
  1. Can export to html and Microsoft Excel
  2. Can import from Clipboard

Mimosa:
    1. Only one window for courses, professors, calendar
    2. No classroom support

FET:

    1. Timetables are shown by week but only shows schedule of one student, instructor, or room at once.

   2. Timetables cannot be edited and all other editing is done through the opening of new edit windows.

   3. A few constraints for "days" exist such as min/max hours daily for instructors and starting days for activities.

   4. Instructors must specify not available times instead of preferred times.
