2.3.1.2. Edit Instructor

In this view, an administrator may edit an instructor's information.



Figure 2.3.1.2a. Edit Instructor window

The only field that cannot be modified is the ID field. If the ID field must be changed then the user must go back and add a new instructor with the new ID field instead.

To complete the add, click on Save. Otherwise, click on Cancel.
