2.7 Installation

The Scheduling Project has two components - a server and a client.

2.7.1 Server Installation

The server component of the Scheduling Project is a Java project. Each department will need to install the server on a computer they control, with the following minimum requirements:

  • Windows XP, Mac OS X 10.6, Linux
  • Java 1.6
  • Open port 80 for non-local use

The Scheduling Project will only be accessible when the server is running, so departments will likely wish to choose a computer that stays on 24/7.

2.7.2 Client Installation

The Scheduling Project client is a web application; no additional software installation is required if the system meets the following minimum requirements:

  • Internet Explorer 8, Mozilla Firefox 3.5, Google Chrome, Safari 5.1
  • 1024x768 or 1280x800 screen resolution
  • A network connection to the server

The client will also be available as a simple desktop application. In addition to the above requirement, the desktop application will require administrator privileges to install.
