Client Interview Data

This directory contains original interview transcripts, including audio files. The files are stored in subdirectories named for the department in which the interview was held. The naming convention for department subdirectory is to use the two to four character directory name, as listed for the campus catalog at 


http://catalog.calpoly.edu/2011cat/course_descr2011.html

See the 308 milestone 2 writeup for the naming conventions of transcript files. The URL is 

http://users.csc.calpoly.edu/~gfisher/classes/308/handouts/milestone2.html#transcript-file
Audio recordings are stored in a DEPT/audio subdirectory, where "DEPT" is the department name. See the preceding URL for the audio file naming conventions. Any pictures taken during the interview are stored in a DEPT/images subdirectory.
