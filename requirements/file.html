2.8. Details of File Commands

In addition to schedule generation, the Scheduler manages the user's scheduling project(s). A Scheduler project is stored in a file with a .sch extension. The Scheduler allows the user to create new projects or open existing ones. The Scheduler also provides commands for the user to edit, save, and print project work. The scenarios in this section cover full details of file-related commands, some of which have been introduced in preceding sections.

      -New
      -Open
      -Open Recent
      -Save
      -Save As
      -Print Preview and Print
      -Close Project
      -Exit
