How to Run

To compile the SchedulerTool, run "make" in the "scheduler/implementation/source/java" directory.

To run the SchedulerTool, run "java -jar scheduler.jar" in the " scheduler/implementation/executables/JVM" directory.

The following table details how various aspects of the SchedulerTool can be accessed/run via the GUI.
