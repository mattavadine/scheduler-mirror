A Generated Schedule Done  
View Blank Schedule Fix The slide title is misleading; it would seem to indicate that the screen should hold a schedule with nothing in it.
View Generated Schedule Fix When a course spans multiple hours, a block with it's name appears in each row it spans; this could be misinterpreted to mean that multiple sections of the course are being scheduled back to back rather than the time span belonging to one section. When a course spans multiple hours, it's name should appear in one block that spans multiple rows.

View Course Schedule Fix Typo at end of blurb: "...for which many sections...", remove word "many".
Filters View Done  
Filters View, Selected Days Done  
Filters View, Selected Instructor Fix If the user wants to view only a subset of instructors, courses, or locations, it seems cumbersome to have to click on each item they don't want to see. Unless the subset contains more than half of the available items, the user will have to do more clicks than they would if selecting an item showed what they do want to see. This would be a good one for usability personel to look into.
