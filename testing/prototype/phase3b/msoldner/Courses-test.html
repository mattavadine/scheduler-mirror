Acceptance Test Plan

The tabular format of an acceptance test plan is the following:

Case No. User Inputs and Command Expected Output Remarks                   Results                  
1 Add a course to the list. Course gets added to the list. Adds perfectly. Added to the schedule.
2 Save the courses list after an add. Course is saved in the list.   Course is saved in the list.
3 Delete a course from the list. Course gets added to the list. Adds perfectly. Added to the schedule.
4 Save the courses after an delete. Course gets deleted from the list. Didn't work. The course still remains on the list. Course doesn't get deleted.
