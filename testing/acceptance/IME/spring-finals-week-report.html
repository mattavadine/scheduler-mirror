2. Enter Resource Data

  1. The row selection (used for resource deletion/duplication) is strange, I want it to be "selected" only if they click directly on the checkbox, not on the row.

  2. It's not clear to the user whether or not max enrollment was per section or overall for the entire course

  3. Other than that, these seem to be working well.

  4. We should consider putting the defaults for the instructor preferences back to "Not Qualified" because it is the most common answer, compared to "Acceptable".

3. Generate a Schedule

  1. Drag and drop defaults need to be more intuative. When the pop-up comes up, it should have defaults that don't give me an error message if I just press "OK".

4. Edit the Schedule

  1. Generate still spits out "gross looking" schedules with lots of classes in a single time spot

  2. When editing, by default it should give a start time and and end time that amounts to an hour difference, not a 1/2 hour difference.
