2. Enter Resource Data

Overall, I was able to enter the necessary data in order to generate a schedule, with the following positives and negatives.

Things that worked:

  • Tabbed data entry
  • Ordering the columns by clicking on the top row
  • Saving, exiting, and reopening a schedule with same data

Issues:
  • Had to double click Day Combinations column in order to see checkboxes
  • After selecting a Day Combo, the column would compress and I would have to double click again
  • Reopening the schedule results in all data being gone without force saving. *I thought we were autosaving?
  • Unable to edit text in a cell that has already been entered. Only able to replace existing text
  • Unable to delete a row. I don't see a way to do this
  • Unable to save association for a class. Appears as "None" after saving
  • Missing day combination for MTWF
  • When tabbing to input values, the view jumps to the top of the page. Need to scroll down to see where I am inputting values
  • I do not see a way to support ACTivities that do not have times associated with them, so I have left them out of the course resources
  • Had a 500 error, failed to update server
  • Update failed: 502, Bad Gateway
  • After these errors occurred, saving did not work. I was told data had saved, but upon reopening, it had not been
  • It is unclear how to crosslist courses. I do not see how to list MU 171 and MU 371 as the same course to be scheduled at the same time
  • Shift+tab makes new row instead of going back a tab
  • Scrolling sideways while the column headers are moving down with the top of the page results in the columns moving, but the headers staying fixed
  • Needed to save and close document to see changes persist for instructor prefs
