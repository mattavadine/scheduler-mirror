Finals Week Acceptance Test Report for the ENGL Department



1. Create, Save, and Re-Open a New Schedule

All actions performed successfully. Just a tip, it should have a confirmation dialog when tyring to delete a schedule.


2. Enter Resource Data

This is actually working really well. My only beef is that it allows me to save rows that have errors. It also allows me to save empty rows. Oh, and also, if I select multiple rows, the dialog should use "these courses" instead of "this course".


3. Generate a Schedule

When I navigated to this page the first time, the calendar wouldn't appear properly. So I logged off and logged back in. It worked fine after this. However, the calendar page does not respect the "scheduleable" property on the resources.


4. Edit the Schedule

Drag and drop works like a dream.


5. Save, Exit, Re-Open Schedule

All actions performed successfully.


6. Quality Measures

Usability is sorely lacking on this application. The interface is clunky and hard to use.


7. Other Issues

If you leave the window idle for a while dialogs about "failed to send updates to server" start to appear. https://scheduler.atlassian.net/browse/SCHED-240 I would NOT use this application.
